This Site

Built from scratch, I used Foundatin CSS as a jumping point due the fact I find it to be lighter than Bootstrap, but also being available in sass I was able to only include the base elements necessary.

A number of useful bits and bobs found in modern templates are also used, such as the ubiquitous jQuery, modernizr, conditionizr, fitText and normalize.css.

The site will use CSS3 transitions where available and fall back to jQuery based animation when it's not. Cross browser compatiable, it's responsive with a focus on ensuring readability at various viewport sizes. Also, as I've hosted on GitHub, the source can be found in my repo.

And finally all this was built on my editor of choice, Sublime with the help of CodeKit. Who would've thought not having to refresh pages myself all the time would be so amazing.
