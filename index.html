This Site

Built from scratch, Foundation CSS as a jumping point not only because I find it lighter than Bootstrap, but also the sass version means only the necessary components are used.

Other useful bits and bobs include the ubiquitous jQuery, modernizr, conditionizr, fitText, normalize.css, Google Web Fonts (Roboto, Bree Serif, Open Sans) & entypo for icons.

The site will use CSS3 transitions where available and fall back to jQuery based animation when it's not. Cross browser compatible, it's responsive with a focus on ensuring readability at various viewport sizes. Also, as I've hosted on GitHub, the source can be found in my repo.

And finally all this was built in my favourite editor, Sublime. CodeKit was also very useful to handle build functions. It also auto-refreshes on changes, which although minor is pretty amazingly useful.
