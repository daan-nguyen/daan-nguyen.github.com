Other Tidbits.

Built from scratch using HTML5 + CSS3 and some Javascript, it’s responsive, relatively cross-browser (farewell IE8 and lesser friends) and uses CSS3 transforms/transitions to maintain animation performance. Bower handles component management (only FastClick and Normalize.css really), TypeKit for webfonts and Grunt is used for building the SCSS (using SASS and Compass) along with some other tasks such as JSHinting, concatenation and minification.

For the Javascript I opted to go vanilla, though a side effect of this being smooth scrolling anchor links were no longer a simple jQuery animate call. Despite much google-fu, I never did quite land on a solution that felt right and ended up rolling my own code, smoothie-scroll.js, which I've made available. Source for this particular page also be found on GitHub.

NB: I later discovered Smooth Scroll which pretty closely fits my needs, except it relies on data attributes to act as a trigger which isn’t quite what I was after anyway.
