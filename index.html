This Site

Built from scratch using HTML5 + CSS3, Foundation CSS was used as a jumping point not only because I find it lighter than Bootstrap, but also the sass version means only the necessary components imported.

Other useful bits and bobs include the ubiquitous jQuery, modernizr, conditionizr, fitText, normalize.css, Google Web Fonts (Roboto, Bree Serif, Open Sans) & entypo for icons.

Cross browser compatible, it's responsive with a focus on ensuring readability at various viewport sizes. Also, as I've hosted on GitHub, the source can be found in my repo.

And finally all this was built in my favourite editor, Sublime. CodeKit was also very useful to handle build functions. It also auto-refreshes on changes, which although minor is pretty amazingly useful.
