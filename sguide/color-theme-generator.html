Color theme switcher

Description

This page describes how to use the mixin .generate-theme(...) in our LESS files. This mixin will allow us to manage all foreground, background and border colours from a centralized place. The mixin is defined in the themes.less file.

Configuration

.generate-theme(...) uses the following lists to configure available themes and colors:

Theme List:
@themes: 'default', 'green', 'orange', 'blue', 'magenta', 'purple', 'grey', 'turquoise';

Theme Colors:
@grey-colors: #8f8f8f, #757575, #575757, #4f4f4f, #363636; @orange-colors: #ff9f33, #ff8700, #f05a00, #ab004c, #780035; @green-colors: #bcfb00, #96c800, #009b46, #005a28, #002711; @turquoise-colors: #00eccc, #00b9a0, #009182, #004b46, #001816; @blue-colors: #33c3ff, #00b4ff, #0099d2, #004d9d, #00346a; @purple-colors: #bab3e8, #968cdc, #644bc8, #46008c, #2c0059; @magenta-colors: #ff5baf, #ff2896, #a0148c, #640064, #310031;
@default-colors: @blue-colors;

Mixin

Definition:

.generate-theme(@mode; @rest...) { ... }

Parameters:

@mode: A string to specify the mode, it could be s or single when you want to generate the color themes for one css rule (e.g only for background-color) or it could be m or multi when you want to generate the color theme for more than one css rule at the same time (e.g. for color, background-color, border-color).

@rest: A list of CSS statements to be configured. CSS statements contains a comma separate list of configuration arguments and they are separated from other CSS statements by semicolons. See the following section to understand the structure of a CSS statement.

CSS statements:

CSS statements can have different comma separate list with configuration arguments as described below:

@property, @shade : where the options for each could be as follow.

@property : [ color | border-color | background-color ]
@shade : [ white | lighter | light | default | dark | darker | black ]

Examples:
.generate-theme(s; color, lighter);
.generate-theme(m; color, lighter; background-color, default);

@property, @function, @shade, @percentage: where the options for each could be as follow.

@property : [ color | border-color | background-color ]
@function : [ lighten | darken ]
@shade : [ white | lighter | light | default | dark | darker | black ]
@percentage : 0-100%

Examples:
.generate-theme(s; color, lighten, dark, 10%);
.generate-theme(m; background-color, darken, 50%; color, dark);

@property, @transition, @fromShade, @toShade: where the options for each could be as follow.

@property : [ background ]
@transition : [ linear-gradient ]
@fromShade : [ white | lighter | light | default | dark | darker | black ]
@toShade : [ white | lighter | light | default | dark | darker | black ]

Examples:
.generate-theme(s; background, linear-gradient, light, dark);
.generate-theme(m; background, linear-gradient, dark, white; color, dark);

NOTE: Try to use the multiple @mode option in .generate-theme(...) as much as possible, this avoids the generation of too many CSS rules by LESS.

Instead of doing this:
.color-theme.fg-color-white {
  .generate-theme(s; background-color, default);
  .generate-theme(s; color, white );
}

Do this:
.color-theme.fg-color-white {
  .generate-theme(m; background-color, default; color, white );
}

Examples

Foreground color


WHITE
.color-theme.fg-color-white { .generate-theme(m; color, white; background-color, default); }
LIGHTER
.color-theme.fg-color-lighter { .generate-theme(s; color, lighter); }
LIGHT
.color-theme.fg-color-light { .generate-theme(s; color, light); }
DEFAULT
.color-theme.fg-color-default { .generate-theme(s; color, default); }
DARK
.color-theme.fg-color-dark { .generate-theme(s; color, dark); }
DARKER
.color-theme.fg-color-darker { .generate-theme(s; color, darker); }
BLACK
.color-theme.fg-color-black { .generate-theme(s; color, black); }

Background color


WHITE
.color-theme.bg-color-white {
   .generate-theme(m; background-color, white; color, default); }
LIGHTER
.color-theme.bg-color-lighter {
   .generate-theme(m; background-color, lighter; color, white); }
LIGHT
.color-theme.bg-color-light {
   .generate-theme(m; background-color, light; color, white); }
DEFAULT
.color-theme.bg-color-default {
   .generate-theme(m; background-color, default; color, white); }
DARK
.color-theme.bg-color-dark {
   .generate-theme(m; background-color, dark; color, white); }
DARKER
.color-theme.bg-color-darker {
   .generate-theme(m; background-color, darker; color, white); }
BLACK
.color-theme.bg-color-black {
   .generate-theme(m; background-color, black; color, white); }

Border color


WHITE
.color-theme.border-color-white { .generate-theme(s; border-color, white); }
LIGHTER
.color-theme.border-color-lighter { .generate-theme(s; border-color, lighter); }
LIGHT
.color-theme.border-color-light { .generate-theme(s; border-color, light); }
DEFAULT
.color-theme.border-color-default { .generate-theme(s; border-color, default); }
DARK
.color-theme.border-color-dark { .generate-theme(s; border-color, dark); }
DARKER
.color-theme.border-color-darker { .generate-theme(s; border-color, darker); }
BLACK
.color-theme.border-color-black { .generate-theme(s; border-color, black); }

Background gradients


WHITE
.color-theme.bg-color-gradient-white { .generate-theme(s; background, linear-gradient, white, black); }
LIGHTER
.color-theme.bg-color-gradient-lighter { .generate-theme(s; background, linear-gradient, lighter, black); }
LIGHT
.color-theme.bg-color-gradient-light { .generate-theme(s; background, linear-gradient, light, black); }
DEFAULT
.color-theme.bg-color-gradient-default { .generate-theme(s; background, linear-gradient, default, black); }
DARK
.color-theme.bg-color-gradient-dark { .generate-theme(s; background, linear-gradient, dark, black); }
DARKER
.color-theme.bg-color-gradient-darker { .generate-theme(s; background, linear-gradient, darker, black); }
BLACK
.color-theme.bg-color-gradient-black { .generate-theme(s; background, linear-gradient, black, white); }

Lighten/darken options


LIGHTEN
.color-theme.bg-color-func-lighten { .generate-theme(s; background-color, lighten, black, 80%); }
DARKEN
.color-theme.bg-color-func-darken { .generate-theme(s; background-color, darken, white, 10%); }
