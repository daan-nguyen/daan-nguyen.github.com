Overview

Carousels consist of a few simple elements:

  • A container - In this case it is the .carousel-banner element.
  • Slides - A set of <div> elements that act as the individual slides for the carousel. These can contain of either a simple <img> or custom html.
  • Options - A variety of data-* options can be supplied to modify the carousel.

Other notes:

  • Carousels will fit the maximum width available in the container they are placed. This means you can use a 50/50 split column and place two carousels side by side and they will fill the space accordingly.
Options

All of the following options can be included on the container element via data-* attributes.

Name Type Default Description
autoplay boolean false Enables auto looping of slides. Enabling autoplay will also automatically enable infinite.
autoplay-speed int 3000 Sets the delay between automatic changing of slides in ms when autoplay is enabled.
arrows boolean true Enables the previous and next arrows.
dots boolean false Enables the navigation dots.
fade boolean false Changes the slide transition from sliding to fading between slides.
infinite boolean false Enables infinite looping of slides, by default moving to the next slide is disabled when the last slide is reached.
slides-to-show int 3 The default value of 3 is only enabled when the .carousel-multi class is added to the .carousel container.
initial-slide int 0 Slide to start on.
center-mode boolean false Enables centered view with partial prev/next slides. Use with odd numbered slidesToShow counts.
center-padding string 8.33% Side padding when in center mode (px or %)
mobile-only boolean false Enables the carousel for mobile only
mobile-[property] inherit inherit Set a property only for mobile

The following are class options that can be attached to various elements to affect the visual presentation.

Class Attaches To Description
.carousel-border .carousel Enables a border around the .carousel.
.carousel-border-theme .carousel Enables a border around the .carousel using the set theme colour.
.bg-colour-light .carousel > div Sets the background colour on the slide to the light theme colour.
.bg-colour-dark .carousel > div Sets the background colour on the slide to the dark theme colour.
.bg-colour .carousel > div Sets the background colour on the slide to grey.
.bg-colour-gradient .carousel > div Sets the background colour on the slide to a gradient going from dark to light.
.bg-grey-gradient .carousel > div Sets the background colour on the slide to a gradient going from grey to light grey.

Multi Slide Carousel

The Multi Slide Carousel provides a carousel that will show multiple slides at the same time. The number of slides can be set from the minimum of 2 to a value no higher than the number of slides available by using the data-slides-to-show attribute.

Note the addition of the .carousel-multi class to the container .carousel element that will set the default slides-to-show value and some other Multi Carousel specific settings.

The Gallery Carousel combines the standard Banner Carousel with the Multi Slide Carousel in order to provide a main gallery image with scrollable thumbnails below it. A .carousel-gallery class is added to the main container. Two lists of slides are then created using the .carousel-gallery-main and .carousel-gallery-list to hold primary and thumbnail images.

Options such as data-slides-to-show are applied to the required carousel specifically (.carousel-gallery-list for example has data-slides-to-show set).
