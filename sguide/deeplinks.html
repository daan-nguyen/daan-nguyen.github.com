Deep linking is used to link to pages with dynamic elements such as Tabs, Switch, Accordions and be able to specify the state of the page as part of the URL. This works for linking to new pages (eg. from abc.html to def.html) and within the page itself via a simple href="#id" link.

Deep linking consists of 2 things:

  • #anchors - standard page anchors, this will cause the page to jump to this specific anchor. All #anchor links will automatically scroll to their targets.
  • --id1.id2.id3 - a list of IDs prefixed by a double hyphen (--) representing various content containers (tabs, switches, accordions etc) can be supplied as a period separated list (id1.id2.id3) to ensure they are active at page load.
Basics

Standard link to an onpage anchor, this will result in the page smooth scrolling to the target specified. If the target is a tab, switch or accordion then it will scroll to the tab and also activate it.

<a href="anotherpage.html#abc">Link to another page</a> <a href="#def">Link on same page</a>

Activate a single/multiple content elements by supplying a period seperated ID list --id1.id2.id3.

<a href="anotherpage.html#--id1.id2.id3">Link to another page</a> <a href="#--id1.id2.id3">Link on same page</a>

These can be combined, always ensure the #anchor is supplied before the list of IDs used to activate the state of content (--id1.id2.id3). Note: If you are using this in combination with a querystring, ensure the # follows after the querystring like so page.html?query=string#abc.

<a href="anotherpage.html#abc--id1.id2.id3">Link to another page</a> <a href="#def--id1.id2.id3">Link on same page</a> Jumbotron

Activating jumbotron differs a little bit to the basic activation of other components above. To activate a jumbotron slide you must add the ID jumbotron and the slide number required : #--jumbotron-3. So to activate the second slide of jumbotron, you would use:

<a href="anotherpage.html#--jumbotron-2">Open new page and activate slide 2</a> <a href="#--jumbotron-2">Link on same page</a> Demo
